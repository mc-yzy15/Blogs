# Yzy15's Blog

基于 [Hexo](https://hexo.io/) + [Butterfly](https://butterfly.js.org/) 搭建的个人博客。

## 链接

- 🌐 博客地址：[blogs.yzy15.dpdns.org](https://blogs.yzy15.dpdns.org)
- 🏠 个人主页：[yzy15.dpdns.org](https://yzy15.dpdns.org)

## 本地运行

```bash
pnpm install
pnpm server
```

## 部署

推送到 `master` 分支后，GitHub Actions 自动构建并部署到 `gh-pages`。
